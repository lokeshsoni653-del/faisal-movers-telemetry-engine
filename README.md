# 🚌 Faisal Movers: Intercity Fleet Telemetry & Dispatch Engine

## 📌 Operational Overview
This repository contains a live, real-time dispatch and telemetry dashboard prototyped for intercity fleet logistics. Built entirely in Python utilizing Streamlit and Folium, the architecture replaces manual transit tracking with an automated, map-based data pipeline.

The system simulates and visualizes dynamic fleet ETAs, fuel burn rates, and terminal cargo capacities across central Pakistan's M-4 and M-5 motorways, providing dispatchers with instant tactical visibility to reduce supply chain latency and optimize route efficiency.
